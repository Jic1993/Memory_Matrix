# Memory_Matrix

## 概述
上大二的时候发现一个叫[Lumosity](https://www.lumosity.com )的很有意思的网站， 类似脑力版的[Keep](http://www.gotokeep.com/)，每天会向用户推送大概5个益智小游戏，帮助用户保持脑力训练。
Memory Matrix是其中一个记忆类的游戏，个人非常喜欢。
